# Music Gallery Web

A simple, responsive music album gallery built using **HTML, CSS, JavaScript, and XML**. The application displays album covers along with basic metadata, providing a clean and user-friendly interface for browsing music collections.

## Overview

This project focuses on front-end development fundamentals and data handling using XML. Music album details are dynamically loaded and rendered on the webpage, demonstrating separation of data and presentation with a lightweight, responsive design.

## Key Features

* Responsive music album gallery
* Album cover and metadata display
* XML-based data storage and parsing
* Clean and minimal UI
* Lightweight and fast loading

## Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling and responsiveness
* **JavaScript** – Dynamic content rendering
* **XML** – Music album data storage

## How It Works

1. Album data is stored in an XML file.
2. JavaScript parses the XML data.
3. Album information is dynamically displayed on the webpage.
4. CSS ensures responsive and clean layout across devices.
